# Rushing Wind App

App movil para el movimiento **Rushing Wind** — una Biblia moderna para Generacion Z con devocional integrado y plataforma de eventos/boleteria para iglesias y organizadores cristianos.

---

## Sobre el proyecto

Rushing Wind es un movimiento cuyo objetivo es promover, propagar y difundir el evangelio. La app es la herramienta digital central de ese movimiento.

La plataforma se compone de **dos aplicaciones**:

- **App del Usuario** — Consumir la Biblia, devocionales, eventos, contenido de creadores y gamificacion.
- **App del Operador** — Crear y gestionar eventos, boleteria y comunicacion con el equipo de Rushing Wind.

## Publico objetivo

- Generacion Z como audiencia principal
- Iglesias y comunidades cristianas de Norteamerica y Latinoamerica
- Creadores de contenido cristiano (predicadores, speakers, artistas)

## Modulos

| Modulo | Descripcion | Prioridad |
|--------|-------------|-----------|
| Biblia | Lectura completa (66 libros), progreso, experiencia moderna | Alta |
| Eventos / Boleteria | Plataforma para publicar eventos y vender boletos | Alta |
| Devocional | Contenido premium segmentado por temas | Media |
| Gamificacion | Puntos, rachas, progreso y logros (transversal) | Media |
| Growth Tracks | Caminos de crecimiento espiritual por dias | Media-Baja |
| Articulos Biblicos | Enciclopedia biblica con articulos de speakers | Media-Baja |
| Guest / Podcast | Contenido de invitados en formato episodio | Baja |
